# 内存管理
## 一、当前内存管理流程

整个代码全部使用值语义（Value Semantics），没有任何裸指针，依赖三种机制自动管理内存：
```
stack 分配               → int, bool 等基础类型
vector<T> 的 RAII        → 出作用域自动销毁堆内存
返回值优化（RVO/NRVO）   → 避免大 vector 的拷贝
```

### 主要算法的生命周期

以 `generateOptimalGroups()` 为例：

```
函数进入
│
├─ jSubsets       vector<vector<int>>  C(n,j) 个 j-子集，每个大小 j
├─ allSSubs       vector<vector<int>>  C(n,s) 个 s-子集（临时中间量）
├─ subIndex       map<vector<int>,int> s-子集 → 全局编号（查找表）
├─ globalSubs     vector<vector<int>>  去重后的全局 s-子集
├─ jToSubs        vector<vector<int>>  j-子集 i → 其 s-子集编号列表
├─ kCandidates    vector<vector<int>>  C(n,k) 个 k-候选组
├─ kCoversS       vector<vector<int>>  k-候选 c → 其覆盖的 s-子集编号列表
│
├─ runGreedy lambda（调用 500 次，每次重新分配）：
│   ├─ usedCand   vector<bool>         候选是否已用
│   ├─ covState   vector<bool>         s-子集是否已覆盖
│   ├─ jCnt       vector<int>          每个 j-子集当前覆盖数
│   ├─ jSat       vector<bool>         j-子集是否已满足
│   └─ 内层循环每轮：coversThis vector<bool>(numSubs) ← 热点！
│
└─ 函数返回 result（move 语义）
```

## 内存遇到的问题
### 第一次修改以及方案）
#### 问题 1（最严重）：热路径内反复分配 `coversThis`
分配次数估算：
- 每次 `runGreedy`：候选数 × 贪心轮数 次分配
- 调用 500 次重启 → 总分配 = `500 × C(n,k) × 轮数`
- 对 n=8（C(8,6)=28）：约 500 × 28 × 6 ≈ 84,000 次 heap 分配/释放

#### 问题 2：500 次重启每次都重新分配 4 个向量
```
auto runGreedy = [&](std::mt19937& rng) {
    std::vector<bool> usedCand(...);  // 500次 × 每次新建
    std::vector<bool> covState(...);
    std::vector<int>  jCnt(...);
    std::vector<bool> jSat(...);
```
这4个向量可以复用

#### 问题 3：`subIndex` 用 `map<vector<int>, int>` 作为查找表
`std::map` 用 `vector<int>` 做键，每次查找需要 O(s × log N) 次元素比较，还有红黑树节点的堆分配开销。

####  问题 4：`allSSubs` 是冗余中间量

```
// 第 264 行
std::vector<std::vector<int>> allSSubs = enumerate(samples_, s_);
// 第 283 行：allSSubs 的内容全部转移到 globalSubs，之后 allSSubs 废弃
```
`allSSubs` 和最终的 `globalSubs` 内容相同，这次分配可以消除。
#### 问题 5：`checkAllSatisfied` 没有复用预计算结果 
```
// 第 419 行，后向消除中
for (int si = 0; si < numSubs; ++si) {
    if (intersectSize(globalSubs[si], g) == s_)  // 重新计算！
```
`kCoversS` 已经预计算了每个 k-组覆盖哪些 s-子集，但 `checkAllSatisfied` 没有用，而是重新调用 `intersectSize`。

## 核心算法

算法思路：
* Universe（待覆盖元素）= 所有 (j-子集 J, s-子集 S⊆J) 的有序对
* Selector（候选集合） = 所有 C(n,k) 个 k-子集
* k-组 K 覆盖有序对 (J, S) ⟺ S ⊆ K
* j-子集 J 被"满足" ⟺ 其被覆盖的 s-子集数 ≥ minCover
* 贪心策略：
* 每轮选出令"新增满足 j-子集数"最多的 k-候选组；
* 平局时优先取字典序最小的组。
* 循环直至所有 j-子集均被满足。

```cpp
std::vector<std::vector<int>> SampleSelectSystem::generateOptimalGroups()

{

if (samples_.empty()) {

throw std::runtime_error("尚未选择样本，请先调用 randomSamples() 或 inputSamples()。");

}

  

// ── 1. 枚举所有 j-子集 ──────────────────────────────────────────────────

std::vector<std::vector<int>> jSubsets = enumerate(samples_, j_);

int numJ = (int)jSubsets.size();

  

std::cout << ">> C(" << n_ << "," << j_ << ") = " << numJ

<< " 个 j-子集，minCover=" << minCover_

<< "，每个 j-子集需覆盖 " << minCover_

<< " 个 s-子集（共 C(" << j_ << "," << s_ << ")="

<< C(j_, s_) << " 个）\n";

  

// ── 2. 为每个 j-子集枚举其 s-子集，并建立全局 s-子集索引 ────────────────

// globalSubs: 所有出现的 s-子集（去重），key: s-子集内容 → 全局编号

// jToSubs[i]: 第 i 个 j-子集对应的全局 s-子集编号列表

std::vector<std::vector<int>> globalSubs; // 全局 s-子集内容

std::vector<std::vector<int>> jToSubs(numJ); // j-子集 i → 其 s-子集全局编号

std::vector<bool> subCovered; // 每个全局 s-子集是否已被覆盖

  

// 用 map 做去重索引

std::vector<std::vector<int>> allSSubs = enumerate(samples_, s_);

// 建立快速查找：s-子集内容 → 全局编号

// 因为 allSSubs 已按字典序排好，用二分可加速；这里数量小，线性也可

auto findOrAdd = [&](const std::vector<int>& sub) -> int {

// 二分查找

auto it = std::lower_bound(globalSubs.begin(), globalSubs.end(), sub);

if (it != globalSubs.end() && *it == sub) {

return (int)(it - globalSubs.begin());

}

int idx = (int)(it - globalSubs.begin());

globalSubs.insert(it, sub);

subCovered.insert(subCovered.begin() + idx, false);

// 已插入的索引后移，需要重建 jToSubs——改用 map 更稳健

return -1; // 不直接用此方式，下面用 map

};

(void)findOrAdd; // 抑制未使用警告，下面换成 map 实现

  

// 用 map 建立 s-子集 → 全局编号

std::map<std::vector<int>, int> subIndex;

for (auto& sub : allSSubs) {

if (subIndex.find(sub) == subIndex.end()) {

int idx = (int)globalSubs.size();

subIndex[sub] = idx;

globalSubs.push_back(sub);

}

}

int numSubs = (int)globalSubs.size();

subCovered.assign(numSubs, false);

  

// 填充 jToSubs

for (int i = 0; i < numJ; ++i) {

std::vector<std::vector<int>> sOfJ = enumerate(jSubsets[i], s_);

for (auto& sub : sOfJ) {

jToSubs[i].push_back(subIndex.at(sub));

}

}

  

// ── 3. 初始化满足状态 ───────────────────────────────────────────────────

// jCoverCount[i] = 第 i 个 j-子集已被覆盖的 s-子集数

std::vector<int> jCoverCount(numJ, 0);

std::vector<bool> jSatisfied(numJ, false);

int unsatisfiedCount = numJ;

  

// ── 4. 枚举所有 k-候选组 ────────────────────────────────────────────────

std::vector<std::vector<int>> kCandidates = enumerate(samples_, k_);

std::cout << ">> C(" << n_ << "," << k_ << ") = " << kCandidates.size()

<< " 个 k-候选组\n";

  

// 预计算：每个 k-候选组覆盖哪些全局 s-子集

// kCoversS[c]: 第 c 个 k-候选组覆盖的全局 s-子集编号集合

std::vector<std::vector<int>> kCoversS(kCandidates.size());

for (int c = 0; c < (int)kCandidates.size(); ++c) {

const auto& kg = kCandidates[c];

// 遍历所有全局 s-子集，检查是否 ⊆ kg

for (int si = 0; si < numSubs; ++si) {

if (intersectSize(globalSubs[si], kg) == s_) {

// 全局 s-子集 globalSubs[si] 的 s 个元素全部在 kg 中

kCoversS[c].push_back(si);

}

}

}

  

// （usedCandidate 移入 runGreedy lambda 内部管理）

  

// ── 5. 单次贪心运行（带随机 seed）─────────────────────────────────────────

// 评分策略（边际收益）：

// 对每个未满足的 j-子集 J，候选组 K 的贡献 =

// min(K 为 J 新增的 s-子集覆盖数, J 还差几个才满足)

// 总分 = Σ 所有未满足 J 的贡献

// 当多个候选分数相同时随机选择（用于多次重启）

  

// 单次贪心的 lambda（接收随机引擎用于同分打乱）

auto runGreedy = [&](std::mt19937& rng) -> std::vector<std::vector<int>> {

// 重置状态

std::vector<bool> usedCand(kCandidates.size(), false);

std::vector<bool> covState(numSubs, false);

std::vector<int> jCnt(numJ, 0);

std::vector<bool> jSat(numJ, false);

int remaining = numJ;

std::vector<std::vector<int>> res;

  

while (remaining > 0) {

int bestScore = -1;

std::vector<int> bestCands; // 同分候选索引

  

for (int c = 0; c < (int)kCandidates.size(); ++c) {

if (usedCand[c]) continue;

  

std::vector<bool> coversThis(numSubs, false);

for (int si : kCoversS[c]) coversThis[si] = true;

  

int score = 0;

for (int i = 0; i < numJ; ++i) {

if (jSat[i]) continue;

int need = minCover_ - jCnt[i];

int newCount = 0;

for (int si : jToSubs[i]) {

if (coversThis[si] && !covState[si]) ++newCount;

}

score += std::min(newCount, need);

}

  

if (score > bestScore) {

bestScore = score;

bestCands.clear();

bestCands.push_back(c);

} else if (score == bestScore) {

bestCands.push_back(c);

}

}

  

if (bestCands.empty() || bestScore == 0) break; // 无解保护

  

// 从同分候选中随机选一个（增加多样性）

std::uniform_int_distribution<int> dist(0, (int)bestCands.size() - 1);

int chosen = bestCands[dist(rng)];

  

usedCand[chosen] = true;

res.push_back(kCandidates[chosen]);

  

for (int si : kCoversS[chosen]) covState[si] = true;

for (int i = 0; i < numJ; ++i) {

if (jSat[i]) continue;

int cnt = 0;

for (int si : jToSubs[i]) if (covState[si]) ++cnt;

jCnt[i] = cnt;

if (cnt >= minCover_) { jSat[i] = true; --remaining; }

}

}

return res;

};

  

// ── 多次随机重启，取最优 ────────────────────────────────────────────────

// 对于小规模实例（候选数 ≤ 100），多重启可显著改善结果

const int NUM_RESTARTS = (kCandidates.size() <= 200) ? 500 : 1;

std::mt19937 rng(42u); // 固定基础 seed，保证可复现

std::vector<std::vector<int>> result;

  

for (int r = 0; r < NUM_RESTARTS; ++r) {

auto candidate = runGreedy(rng);

if (result.empty() || candidate.size() < result.size()) {

result = std::move(candidate);

}

}

  

std::cout << ">> 贪心阶段完成（" << NUM_RESTARTS << " 次重启）：共 "

<< result.size() << " 个 k-样本组，开始后向消除...\n";

  

// ── 6. 后向消除（Backward Elimination）──────────────────────────────────

// 检验函数：给定候选 k-组集合，判断是否满足所有 j-子集的 minCover 约束

auto checkAllSatisfied = [&](const std::vector<std::vector<int>>& groups) -> bool {

// 重置

std::vector<bool> cov(numSubs, false);

for (const auto& g : groups) {

// 找到此 k-组覆盖哪些全局 s-子集

for (int si = 0; si < numSubs; ++si) {

if (intersectSize(globalSubs[si], g) == s_) cov[si] = true;

}

}

for (int i = 0; i < numJ; ++i) {

int cnt = 0;

for (int si : jToSubs[i]) if (cov[si]) ++cnt;

if (cnt < minCover_) return false;

}

return true;

};

  

// 反复尝试删除一个 k-组直到无法再删

bool improved = true;

while (improved) {

improved = false;

for (int r = 0; r < (int)result.size(); ++r) {

// 临时去掉第 r 个

std::vector<std::vector<int>> candidate;

candidate.reserve(result.size() - 1);

for (int i = 0; i < (int)result.size(); ++i) {

if (i != r) candidate.push_back(result[i]);

}

if (checkAllSatisfied(candidate)) {

result = std::move(candidate);

improved = true;

break; // 重新从头开始

}

}

}

  

std::cout << ">> 后向消除完成：最终共 " << result.size() << " 个 k-样本组\n";

return result;

}
```
