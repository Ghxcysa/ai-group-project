## 1. 背景与目标

众所周知，过去几年由于 **COVID-19 疫情**、更便捷的互联网接入，以及廉价/低成本的大容量存储设备，所生成的数据量变得越来越巨大。通过互联网与通信线路传输数据非常容易，而数字数据也已广泛应用于生活的各个方面。如今，这些大数据被用于数据挖掘、知识发现、机器学习、统计学习、统计分析以及各类实验。

为了从这些大数据中提取或发现有用的数据、信息或知识，我们通常采用的方法之一是 **最优样本选择（optimal samples selection）**。

在本小组项目中，你需要从大数据中提取一个**样本子集**。为提取该子集（样本），必须确保所提取/选择的样本子集尽可能 **公平、无偏**，同时也尽可能 **最优**。下面提出一种可用的方法。

---

## 2. 问题设定与组合选择流程

- 假设共有 **m 个样本**（45 ≤ m ≤ 54）。
- 从这 m 个样本中，**随机选择 n 个样本**（7 ≤ n ≤ 25）形成一组。这样的 n 样本组共有 **mCn** 组。
- 从某一组 n 样本中，再随机选择例如 **k=6**（4 ≤ k ≤ 7）个样本来形成若干组，因此会有 **nCk** 组（当 k=6 时就是 nC6 组）。
- 在这些 k=6 的样本组中，我们希望通过**只选择其中一部分组**来实现“优化”。

---

## 3. 需要满足的条件

### 条件 1
至少存在 **一个** k 样本组，在其中从 **j 个样本**（s ≤ j ≤ k）里选取了 **s 个样本**（3 ≤ s ≤ 7）。也就是说：

- 当 j=4 时，s=3 或 4  
- 当 j=5 时，s=3、4 或 5  
- 当 j=6 时，s=3、4、5 或 6  
- 当 j=7 时，s=3、4、5、6 或 7
